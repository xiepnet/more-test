标签：!DOCTYPE

说明：指定了 HTML 文档遵循的文档类型定义(DTD)。<!DOCTYPE> 声明位于文档中的最前面的位置，处于 标签之前。该标签告知浏览器文档所使用的 HTML 规范。

HTML 4与xhtml都规定了三种文档类型：Strict、Transitional 以及 Frameset。而html5对它进行了简化处理<!DOCTYPE HTML>

标签没有结束标签！<!DOCTYPE> 对大小写不敏感。
